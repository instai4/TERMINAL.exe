## 🌟 Features 🌟

*   **Authentic Hacker Terminal UI:** Experience a fully functional terminal interface complete with a mesmerizing CRT scanline overlay for that classic cyberpunk vibe. 🖥️
*   **Boot Sequence Animation:** Witness a cool boot-up animation with a progress bar on your first visit – it's like booting up a real system! ⚡
*   **`whoami` Command:** Discover my identity! This command reveals my identity card, a concise bio, and a timeline of my experience.
*   **`ls projects` Command:** Browse all my projects displayed as interactive, clickable cards. Each card includes live demo and GitHub links for easy exploration. 📂
*   **`skills` Command:** Get a visual breakdown of my tech stack, color-coded by category for quick understanding. 🎨
*   **`contact` Command:** Find all my contact channels neatly organized with clickable links. Let's connect! ✉️
*   **Tab Autocomplete:** No more typing headaches! Tab completion is available for all commands, making navigation a breeze. 💨
*   **Command History:** Relive your past commands with arrow key navigation. Access the last 50 commands you've typed. ⏪
*   **AI Fallback:** Don't know a command? No worries! Unknown commands are intelligently handled by Grok, Groq, or Gemini, responding in character within the terminal persona. 🤖
*   **Easter Egg:** Who knows what secrets lie hidden? Try `sudo rm -rf /` if you dare... 😉
*   **Embeddable Widget:** Integrate this cool terminal into your own portfolio as a floating widget via iframe and a Floating Action Button (FAB). 🌐

## 💡 How to Use 💡

1.  **Visit the Terminal:** Head over to the [live demo](https://terminal-exe.vercel.app/)!
2.  **Type Commands:** Start by typing `help` to see a list of all available commands.
3.  **Explore:** Try out key commands like `whoami`, `ls projects`, `skills`, and `contact` to learn more about me.
4.  **AI Interaction:** Type any other command, and watch the AI respond in character, simulating a real terminal interaction.
5.  **Navigation:**
    *   Press `Tab` for command autocompletion.
    *   Use `Arrow Keys` to cycle through your command history.
    *   Press `Ctrl+L` to clear the terminal screen.

## 📥 Install 📥

This is a web application deployed on Vercel. There's no traditional "installation" needed for users.

For developers looking to run this locally or contribute:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/instai4/TERMINAL.exe.git
    cd TERMINAL.exe
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Set up environment variables:** You'll need to set up your API keys for Grok, Groq, and Gemini. Create a `.env.local` file in the root of the project and add your keys:
    ```
    GROK_API_KEY=your_grok_api_key
    GROQ_API_KEY=your_groq_api_key
    GEMINI_API_KEY=your_gemini_api_key
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
5.  **Open in browser:** Navigate to `http://localhost:3000` (or the port specified by your dev server).
